## Getting Started

First, you must install the necessary packages with "npm i".

After that you can run the development server with:

```
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
